# InitArray - Java

Programa simples em Java que cria e preenche um array usando
argumentos passados pela linha de comando.

## Como funciona
O programa recebe três valores:
1. Tamanho do array
2. Valor inicial
3. Incremento

Com esses valores, ele preenche o array e exibe o índice e o valor.

## Exemplo de execução
javac InitArray.java
java InitArray3.InitArray 5 10 2

Saída esperada
Index   Value
    0      10
    1      12
    2      14
    3      16
    4      18
